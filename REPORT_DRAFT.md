# 数字心脏 —— FitzHugh-Nagumo 心肌电传播仿真实验报告

## 1 模型思想

心肌细胞的电活动可用 Hodgkin-Huxley (HH) 方程精确描述，但 HH 模型包含四个非线性微分方程和大量离子通道参数，直接用于组织尺度仿真的计算成本极高。FitzHugh (1961) 与 Nagumo 等 (1962) 分别从数学和电路角度提出了一种二变量简化模型，保留了 HH 模型最核心的两个动力学特征：

- **快变量 $v$（膜电位）**：描述细胞的去极化-复极化过程，具有三次非线性自激特性 $v - v^3/3$，能够产生阈值激发与脉冲式响应。
- **慢变量 $w$（恢复变量）**：代表钾离子通道等缓慢恢复机制，负责在激发后将细胞拉回静息态，决定了不应期的长短。

在二维组织中加入扩散项 $D\nabla^2 v$ 后，模型可再现兴奋波在心肌组织中的传播、碰撞、湮灭和螺旋形成等丰富的时空动力学行为。

## 2 参数含义

| 参数 | 默认值 | 物理意义 |
|------|--------|----------|
| $D$ | 1.0 | 扩散系数，反映细胞间缝隙连接的电耦合强度。$D$ 越大传播速度越快 |
| $\varepsilon$ | 0.08 | 时间尺度分离参数。$\varepsilon$ 越小，$w$ 的恢复越慢，不应期越长 |
| $\beta$ | 0.7 | 恢复变量的偏移量，影响静息点和激发阈值 |
| $\gamma$ | 0.8 | 恢复变量的阻尼系数，影响复极化速率 |
| $dt$ | 0.1 | 时间步长，需满足 $D \cdot dt / dx^2 < 0.25$（二维 CFL 条件） |
| $dx$ | 1.0 | 空间网格步长 |

## 3 数值方法

### 3.1 时间积分
采用显式前向 Euler 法。优点是实现简单、每步仅需一次右端函数求值；缺点是条件稳定，$dt$ 受 CFL 约束。默认参数下 $D \cdot dt / dx^2 = 0.1 < 0.25$，满足稳定性条件。

### 3.2 空间离散
拉普拉斯算子使用经典**五点差分格式**：

$$\nabla^2 v_{i,j} \approx \frac{v_{i+1,j}+v_{i-1,j}+v_{i,j+1}+v_{i,j-1}-4v_{i,j}}{dx^2}$$

边界条件采用 **Neumann（零通量）** 条件 $\partial v / \partial n = 0$，通过 `np.pad(mode='edge')` 构造幽灵单元实现，物理含义为组织边缘不向外散失电流。

## 4 实验设计与结果对比

### 4.1 正常传播（对照组）
- **刺激方案**：在左上角 $5\times5$ 区域施加 $v=1.5$ 的初始脉冲（模拟窦房结起搏）。
- **预期结果**：激发波以近似同心圆/扇形向右下角均匀扩展，波前光滑，波速恒定，经过约 2000 步后全组织完成一次完整的去极化-复极化周期。
- **输出文件**：`normal_heartbeat.gif`

### 4.2 螺旋波再入（实验组 A）
- **刺激方案**：S1-S2 交叉场协议 —— S1 从左侧边缘激发平面波，待波前通过组织中部后（$t=60$），在左下象限施加 S2 刺激，制造断裂波前。
- **预期结果**：断裂端在扩散-恢复竞争下自卷曲形成持续旋转的**螺旋波**（spiral wave），这是房颤/室速最经典的数学模型。
- **输出文件**：`arrhythmia_heartbeat.gif`

### 4.3 传导阻滞（实验组 B）
- **修改**：在组织中部放置一条 $40\times10$ 的"坏死区"（$v=w=0$ 恒定），同时将 $D$ 降低到 0.6。
- **预期结果**：波前遇到坏死带后被迫绕行，在障碍物两端形成衍射尾迹和低电位阴影区，传播不再均匀，类似于心梗后的**传导阻滞**。
- **输出文件**：`arrhythmia_block.gif`

## 5 模型优缺点

### 优点
1. **计算高效**：仅两个变量，适合在普通笔记本上进行大网格、长时间仿真。
2. **定性忠实**：能复现激发、不应期、螺旋波、传导阻滞等心电核心现象。
3. **参数直觉**：每个参数都有清晰的物理对应，便于课堂教学与参数扫描。

### 缺点
1. **动作电位形态失真**：FHN 产生对称脉冲，而真实心肌动作电位具有快速去极化和缓慢平台期，需要更复杂模型（如 Beeler-Reuter、Ten Tusscher）来捕捉。
2. **无离子通道细节**：不区分 Na⁺、K⁺、Ca²⁺ 等离子流，无法研究药物对特定通道的作用。
3. **各向同性假设**：本实现中 $D$ 为标量，未考虑心肌纤维方向导致的各向异性传导。
4. **二维简化**：真实心室为三维厚壁结构，跨壁传播和纤维旋转效应在 2D 中无法体现。
